# Oakland Pride Parade Website

The official website for the Oakland Pride Parade, serving as a central hub for information for all contingents and volunteers. Our mission is to ensure the safe, efficient, and joyous execution of the annual parade by providing all necessary resources in one easy-to-access location.

## Description

This website, built with **Jekyll** and **Bootstrap**, is designed to streamline the management of the Oakland Pride Parade by offering dedicated sections for our two primary user groups: parade contingents and volunteers. It provides essential details, forms, and updates to facilitate smooth participation and operation on the day of the event.

## Key Features

* **Contingent Information:** Provides parade route details, check-in procedures, rules, and downloadable forms for participating groups.
* **Volunteer Resources:** Offers information on volunteer roles, training schedules, day-of-event responsibilities, and contact information for team leads.
* **Safety Guidelines:** A dedicated section outlining all safety protocols and emergency procedures for a secure event.
* **Real-time Updates:** A potential area for live announcements and last-minute changes on the day of the parade.
* **Contact Forms:** Easy-to-use forms for contingents and volunteers to submit questions or concerns.

---

## For Contingents

If you are a registered parade contingent, please refer to the following sections on the website for critical information:

* **Parade Route Map:** A detailed map showing the starting point, route, and dispersal area.
* **Check-in & Staging:** Instructions for where and when your group should check in on the day of the parade.
* **Rules and Regulations:** Important guidelines to ensure the safety and success of all participants.

---

## For Volunteers

Volunteers are the heart of the Oakland Pride Parade. The website offers these resources for your success:

* **Volunteer Roles:** Descriptions of the different roles available and their responsibilities.
* **Volunteer Training:** Information on required training sessions, dates, and locations.
* **Day-of Checklist:** A helpful checklist to ensure you are prepared for your role on parade day.

## Getting Started (for Developers)

This project is built using **Jekyll** and **Bootstrap**. To get the project up and running on your local machine for development and testing, follow these steps.

### Prerequisites

You will need the following software installed:

* Ruby
* Jekyll
* Bundler

### Installation

1.  Clone the repository:
    ```bash
    git clone [https://github.com/your-username/oakland-pride-parade-website.git](https://github.com/your-username/oakland-pride-parade-website.git)
    cd oakland-pride-parade-website
    ```

2.  Install dependencies:
    ```bash
    bundle install
    ```

### Running the App

1.  Start the local development server:
    ```bash
    bundle exec jekyll serve
    ```

2.  The application will be available at `http://localhost:4000` (the default Jekyll port).
